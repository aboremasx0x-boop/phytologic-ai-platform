بوابة دخول موحدة لمنصة التشخيص، التنبؤ، التنبيهات، والمراقبة الوطنية

صفحة دخول احترافية بنفس الهوية الموحدة للمنصة، ومهيأة للعمل الآن بوضع تجريبي محلي، وجاهزة للربط لاحقًا مع أي API حقيقي للمصادقة دون الحاجة لإعادة تصميم الواجهة.

التشخيص الذكي
رفع الصور، تحليل الإصابة، حساب الشدة، والتقرير PDF.
التوقعات الزراعية
طقس حقيقي، توقع حي، مقارنة مناطق، وتحليل AI.
مركز العمليات
متابعة لحظية للتنبيهات والمخاطر على مستوى المناطق.
المنظور الوطني
لوحات وطنية وإدارية ضمن واجهة موحدة ومترابطة.
هذه الصفحة لا تفترض وجود API تسجيل دخول غير موجود. عند إضافة endpoint حقيقي للمصادقة، سيتم تعديل جزء JavaScript فقط مع الإبقاء على نفس التصميم.

تسجيل الدخول

أدخل بيانات الدخول للوصول إلى المنصة. يمكن تشغيل الصفحة الآن بوضع تجريبي، أو ربطها لاحقًا مع API مصادقة حقيقي.
مهم: حسب الكود الذي أرسلته، لا يوجد حاليًا endpoint حقيقي لتسجيل الدخول داخل الـ API. لذلك هذه الصفحة تعمل الآن بطريقة آمنة كواجهة دخول محلية، ويمكن تحويلها لاحقًا إلى مصادقة حقيقية بسهولة.
